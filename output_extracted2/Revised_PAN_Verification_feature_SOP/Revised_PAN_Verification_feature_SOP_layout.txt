Common Reasons for PAN not getting validated
1. Name as per PAN printed card copy contains Surname first
Name as per PAN copy -: Kumare Prashant
Name as per IT database -: Prashant Kumare
2. Name as per PAN contains single letter first name, however IT database contains complete
name
Name as per PAN copy -: G Prem Kumar
Name as per IT database -: Prem Kumar Ganeshan
